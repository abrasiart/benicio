PASSOS PARA PUBLICAR O SITE NO VERCEL:

1. Vá para https://vercel.com
2. Clique em "Add New" > "Project" > "Import Manually"
3. Envie o conteúdo descompactado dessa pasta (não o .zip, mas os arquivos dentro dele)
4. Clique em "Deploy"

O site será publicado e você poderá acessar por um link como: https://cha-do-benicio.vercel.app
